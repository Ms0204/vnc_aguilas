Esta es una carpeta para almacenar imágenes.

Para agregar un logo personalizado:
1. Coloca tu imagen en esta carpeta (logo.png, logo.jpg, logo.svg, etc.)
2. Actualiza la ruta en el archivo base.html

Ejemplo en base.html:
<img src="{% static 'img/tu-logo.png' %}" alt="Logo" class="sidebar-logo-img">

O si prefieres usar un SVG inline o un ícono de Font Awesome, ya está configurado.
